ImageTool
=========

A powerful image batch processing tools

feature
-------

1. drag image files
2. format support jpg,png,gif,jxr,atf,ajpg
3. pack images custom outdata
4. trim alpha
5. scale
6. resize
7. bat file
8. support win,mac,linux
